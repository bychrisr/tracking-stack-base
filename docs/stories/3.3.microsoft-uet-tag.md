## Story

**As a** recipient do tracking stack que roda Microsoft/Bing Ads,  
**I want** o snippet do UET Tag já no template (comentado) com exemplo de evento,  
**so that** possa ativar o tracking Bing sem pesquisar o snippet correto.

## Acceptance Criteria

1. Snippet Microsoft UET adicionado como bloco HTML **comentado** em `examples/lead-form-page/index.html` antes do `</head>`
2. Mesmo snippet adicionado em `examples/sales-page/index.html`
3. Placeholder `SEU_UET_TAG_ID` claramente marcado
4. Comentário inclui exemplo de disparo de evento: `window.uetq.push('event', 'submit_lead_form', {})` no mesmo momento que `/tracker`
5. Comentário orienta onde obter o Tag ID: `ads.microsoft.com → Conversion Tracking → UET Tag`

## Tasks / Subtasks

  - [ ] Adicionar snippet UET como bloco comentado antes do `</head>`
  - [ ] Adicionar exemplo de evento no comentário próximo ao submit do formulário
- [ ] Mesmo procedimento em `examples/sales-page/index.html` (AC: 2)
- [ ] Verificar placeholders e comentários de orientação (AC: 3, 4, 5)
