## Story

**As a** recipient do tracking stack que roda Microsoft/Bing Ads,  
**I want** o snippet do UET Tag já no template (comentado) com exemplo de evento,  
**so that** possa ativar o tracking Bing sem pesquisar o snippet correto.

1. [x] Snippet Microsoft UET adicionado como bloco HTML **comentado** em `examples/lead-form-page/index.html` antes do `</head>`
2. [x] Mesmo snippet adicionado em `examples/sales-page/index.html`
3. [x] Placeholder `SEU_UET_TAG_ID` claramente marcado

5. [x] Comentário orienta onde obter o Tag ID: `ads.microsoft.com → Conversion Tracking → UET Tag`
