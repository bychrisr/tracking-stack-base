## Story

**As a** recipient do tracking stack que roda TikTok Ads,  
**I want** o snippet do TikTok Pixel já no template de exemplo (comentado),  
**so that** possa ativar o pixel client-side sem precisar pesquisar o snippet correto.

## Acceptance Criteria

1. Snippet do TikTok Pixel adicionado como bloco HTML **comentado** em `examples/lead-form-page/index.html` antes do `</head>`
2. Mesmo snippet adicionado em `examples/sales-page/index.html`
3. Placeholder `SEU_TIKTOK_PIXEL_ID` claramente marcado
4. Comentário orienta: "ativar apenas se rodar TikTok Ads — substitua SEU_TIKTOK_PIXEL_ID"
5. Snippet dispara evento `CompleteRegistration` no mesmo momento que o lead é enviado (na lead-form-page) — documentado no comentário como exemplo

## Tasks / Subtasks

- [ ] Ler `examples/lead-form-page/index.html` para mapear onde o pixel Meta está (referência de posicionamento) (AC: 1)
  - [ ] Adicionar snippet TikTok como bloco comentado na mesma região
- [ ] Mesmo procedimento em `examples/sales-page/index.html` (AC: 2)
- [ ] Verificar placeholders e comentários de orientação (AC: 3, 4, 5)
