## Objetivo
Criar o adapter de webhook para a Monetizze, permitindo o rastreamento de vendas desta plataforma no mercado brasileiro.

## Requisitos Técnicos
1. **Webhook Adapter:** Criar `functions/webhook/monetizze/[slug].js`.
2. **Padrão:** Basear a implementação no adapter da Eduzz, adaptando o mapeamento de campos.
3. **Atribuição:** Garantir captura do `trk` (enviado via parâmetro de rastreio da Monetizze).
4. **Configuração Cloudflare:** Adicionar `MONETIZZE_WEBHOOK_SLUG` ao `wrangler.toml.example`.
