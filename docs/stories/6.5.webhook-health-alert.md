## Objetivo
Criar um sistema de alerta automático que notifique caso o sistema não receba webhooks de compra por um período de 24h enquanto há tráfego ativo, indicando uma possível quebra de integração.

## Requisitos Técnicos
1. **Health Check Worker:** Criar script em `functions/scripts/webhook-health.js`.
2. **Lógica:** Verificar se `purchase_log` teve entradas nas últimas 24h se `sessions` teve visitas.
3. **Notificação:** Integrar com ManyChat ou envio de email (via Mailchannels/SendGrid) para o admin.

## Definição de Pronto (DoD)
- [ ] Worker configurado para rodar diariamente via Cron.
- [ ] Notificação enviada com sucesso em ambiente de teste.
- [ ] Documentação de como configurar o alerta em `docs/monitoring.md`.
