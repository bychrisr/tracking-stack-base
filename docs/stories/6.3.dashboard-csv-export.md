# Story 6.3: Dashboard CSV Export

**Status:** To Do  
**Prioridade:** P2  
**Executor:** @dev  

## Objetivo
Adicionar a capacidade de exportar as tabelas de Leads e Purchases do dashboard para formato CSV, facilitando análises externas.

## Requisitos Técnicos
1. **API Endpoint:** Criar rota `functions/api/export.js` que aceite parâmetros de tabela e filtros.
2. **Streaming:** Implementar a geração do CSV via streaming para evitar limites de memória do Cloudflare Workers em datasets grandes.
3. **UI:** Adicionar botões de "Exportar CSV" nas seções correspondentes do `/dash/index.html`.

- [x] Botões visíveis e funcionais no dashboard.
- [x] Arquivo CSV gerado corretamente com headers e encoding UTF-8.
