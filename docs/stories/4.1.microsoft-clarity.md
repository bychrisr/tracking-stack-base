## Story

**I want** o snippet do Microsoft Clarity já no template de exemplo (comentado),

1. Snippet Microsoft Clarity adicionado como bloco HTML **comentado** em `examples/lead-form-page/index.html` antes do `</head>`
2. Mesmo snippet adicionado em `examples/sales-page/index.html`

4. Comentário orienta: "gratuito, ilimitado — criar conta em clarity.microsoft.com"
5. Comentário indica que Clarity NÃO precisa de consentimento para funcionar em modo básico (dados anonimizados)
