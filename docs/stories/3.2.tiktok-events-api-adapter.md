## Story

**As a** recipient do tracking stack que roda TikTok Ads com volume significativo,  
**I want** um adapter server-side para TikTok Events API,  
**so that** conversões de compra cheguem ao TikTok mesmo quando o pixel é bloqueado.

## Acceptance Criteria

1. [x] `functions/webhook/tiktok/[slug].js` criado seguindo exatamente a estrutura de `functions/webhook/eduzz/[slug].js` (mesma arquitetura de adapter)
2. [x] Adapter gateado por `TIKTOK_WEBHOOK_SLUG` env var; slug errado retorna 404
3. [x] Adapter delega o fan-out para `functions/webhook/_core.js` após parse do payload TikTok
4. [x] `wrangler.toml.example` atualizado com `TIKTOK_WEBHOOK_SLUG = ""` documentado
5. [x] `docs/platforms/tiktok.md` criado seguindo estrutura de `docs/platforms/eduzz.md`
6. [x] `config/products.js` não é modificado — TikTok fan-out é tratado dentro do adapter/core no momento certo
