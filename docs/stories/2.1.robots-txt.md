# Story 2.1: Criar robots.txt com AI crawlers liberados

**Status:** Ready for Review  
**Epic:** 02 — SEO / AEO / Visibilidade IA  
**Sprint:** 1  
**executor:** @dev  
**quality_gate:** @qa  

## Story

**As a** recipient do tracking stack,  
**I want** um `robots.txt` com permissão explícita para AI crawlers,  
**so that** meu site apareça em respostas de IAs como ChatGPT, Perplexity e Gemini.

## Acceptance Criteria

1. Arquivo `robots.txt` criado na raiz do projeto
2. Contém `Allow: /` para `User-agent: *`
3. Contém permissões explícitas para: `GPTBot`, `anthropic-ai`, `Google-Extended`, `PerplexityBot`, `Applebot-Extended`
4. Contém linha `Sitemap:` com placeholder `https://seusite.com/sitemap.xml`
5. Comentário no arquivo explica o propósito para o recipient

## Tasks / Subtasks

- [x] Verificar se `public/` existe no projeto ou se o diretório correto é outro (AC: 1)
  - [x] Concluí que a raiz `.` é o diretório de saída conforme `wrangler.toml.example`.
- [x] Criar `robots.txt` na raiz com todos os user-agents listados (AC: 2, 3, 4)
- [x] Adicionar comentário orientando o recipient a atualizar a URL do Sitemap (AC: 5)
