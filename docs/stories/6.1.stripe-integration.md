# Story 6.1: Integração Stripe (Épico)

**Status:** To Do  
**Prioridade:** P1  
**Executor:** @dev  

## Objetivo
Implementar a integração com a Stripe como plataforma de vendas, suportando rastreamento server-side via webhook com verificação de assinatura desde o primeiro dia.

## Requisitos Técnicos
1. **Webhook Adapter:** Criar `functions/webhook/stripe/[slug].js`.
2. **Assinatura:** Implementar verificação HMAC-SHA256 usando o header `Stripe-Signature` e a secret `STRIPE_WEBHOOK_SECRET`.
3. **Eventos:** Tratar `checkout.session.completed` e `payment_intent.succeeded`.
4. **Mapeamento `trk`:** Extrair o identificador de atribuição `trk` do campo `metadata` da sessão de checkout.
5. **Configuração de Produto:** Atualizar `config/products.js` para suportar `price_id` ou `product_id` (strings) em vez de apenas IDs numéricos.
6. **Configuração Cloudflare:** Adicionar `STRIPE_WEBHOOK_SECRET` e `STRIPE_WEBHOOK_SLUG` ao `wrangler.toml.example`.
7. **Documentação:** Criar `docs/platforms/stripe.md`.

## Definição de Pronto (DoD)
- [ ] Adapter criado e validado com payloads reais (ou mocks fiéis).
- [ ] Requests sem assinatura válida são rejeitados (401).
- [ ] PII (email, telefone) extraído e normalizado conforme padrão do core.
- [ ] Atribuição `trk` funcionando via metadata.
- [ ] `docs/platforms/stripe.md` detalhando a configuração no dashboard da Stripe.
