# Story 1.2: Configurar Consent Mode v2 (4 sinais)

**Status:** Draft  
**Epic:** 01 — LGPD / Consent Mode v2  
**Sprint:** 1  
**executor:** @dev  
**quality_gate:** @qa  

## Story

**As a** recipient do tracking stack,  
**I want** os 4 sinais do Google Consent Mode v2 configurados nos templates,  
**so that** o Google consiga modelar conversões de usuários que recusam cookies, preservando a performance das campanhas.

## Acceptance Criteria

1. Bloco `gtag('consent', 'default', {...})` adicionado em `examples/lead-form-page/index.html` **antes** do carregamento do `gtag.js`, com os 4 sinais: `ad_storage`, `analytics_storage`, `ad_user_data`, `ad_personalization` setados como `'denied'`
2. Mesmo bloco adicionado em `examples/sales-page/index.html`
3. Comentário explica que o CMP deve chamar `gtag('consent', 'update', {...})` após aceite do usuário
4. Documentação em `docs/` (ou comentário in-line) referencia os dois modos: básico (dispara sempre com sinais) e avançado (bloqueia até consentimento)
5. Nenhuma chamada `gtag('event', ...)` ou pixel fire ocorre antes do bloco de consent default nos templates

## Tasks / Subtasks

- [ ] Ler `examples/lead-form-page/index.html` para verificar posição atual do gtag (AC: 1)
  - [ ] Inserir bloco `gtag consent default` antes do `<script src="gtag.js">` ou `<script src="/scripts/gtag.js">`
- [ ] Mesmo procedimento em `examples/sales-page/index.html` (AC: 2)
- [ ] Adicionar comentário explicando atualização do consent após aceite (AC: 3)
- [ ] Verificar ordem final: CMP → consent default → gtag.js → pixels (AC: 5)

## Dev Notes

**Bloco Consent Mode v2 (template):**
```html
<!-- Google Consent Mode v2 — 4 sinais obrigatórios
     Modo básico: dispara gtag com sinais 'denied', Google modela o restante.
     Modo avançado: CMP chama gtag('consent','update',{...}) após aceite.
     DEVE vir antes do carregamento do gtag.js -->
<script>
  window.dataLayer = window.dataLayer || [];
  function gtag(){dataLayer.push(arguments);}
  gtag('consent', 'default', {
    'ad_storage': 'denied',
    'analytics_storage': 'denied',
    'ad_user_data': 'denied',
    'ad_personalization': 'denied'
  });
</script>
```

**Ordem correta de carregamento:**
1. Snippet CMP (Story 1.1)
2. Bloco `gtag consent default` (esta story)
3. `<script src="/scripts/gtag.js?id=...">` ou CDN do gtag
4. Meta Pixel
5. Demais scripts

**Dependência:** Story 1.1 deve estar concluída antes desta.

**Arquivos afetados:**
- `examples/lead-form-page/index.html`
- `examples/sales-page/index.html`
