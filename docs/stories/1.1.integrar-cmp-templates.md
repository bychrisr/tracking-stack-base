# Story 1.1: Integrar CMP nos templates de exemplo

**Status:** Draft  
**Epic:** 01 — LGPD / Consent Mode v2  
**Sprint:** 1  
**executor:** @dev  
**quality_gate:** @qa  

## Story

**As a** recipient do tracking stack,  
**I want** o CMP (CookieYes) integrado nos templates de exemplo,  
**so that** possa ir ao ar com conformidade LGPD sem precisar descobrir como integrar o banner de consentimento por conta própria.

## Acceptance Criteria

1. Snippet do CMP (CookieYes) adicionado em `examples/lead-form-page/index.html` no `<head>`, **antes** do `gtag.js` e do pixel Meta
2. Mesmo snippet adicionado em `examples/sales-page/index.html` na mesma posição
3. Placeholder `SEU_COOKIEYES_ID` claramente comentado para substituição pelo recipient
4. Comentário no HTML explica: "instalar antes de qualquer pixel — LGPD obrigatório"
5. Link para `tracking-playbook.md` seção 11 incluído no comentário como referência

## Tasks / Subtasks

- [ ] Ler `examples/lead-form-page/index.html` para mapear posição exata de inserção (AC: 1)
  - [ ] Identificar onde `gtag.js` está carregado no `<head>`
  - [ ] Inserir snippet CookieYes imediatamente antes do `gtag.js`
- [ ] Ler `examples/sales-page/index.html` para mesma operação (AC: 2)
  - [ ] Verificar ordem de carregamento de scripts na page
  - [ ] Inserir snippet CookieYes antes dos pixels
- [ ] Validar que placeholder e comentário estão claros (AC: 3, 4, 5)
- [ ] Verificar que nenhum script de tracking aparece antes do CMP nos dois arquivos

## Dev Notes

**Snippet CookieYes (template):**
```html
<!-- CMP — LGPD obrigatório. Substitua SEU_COOKIEYES_ID pelo ID da sua conta.
     Referência: tracking-playbook.md seção 11
     DEVE vir antes de qualquer pixel ou gtag.js -->
<script id="cookieyes" type="text/javascript"
  src="https://cdn-cookieyes.com/client_data/SEU_COOKIEYES_ID/script.js">
</script>
```

**Posicionamento crítico:** O snippet do CMP DEVE ser o primeiro `<script>` no `<head>`. Qualquer pixel que carregue antes invalida o propósito do consent.

**Arquivos afetados:**
- `examples/lead-form-page/index.html`
- `examples/sales-page/index.html`
